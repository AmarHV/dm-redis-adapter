dm-redis-adapter by whoahbot

A DataMapper adapter for the redis key-value database

This is an experimental datamapper adapter for the Redis key-value database. Please be aware that this is very alpha quality software! It is not yet ready for production use.

Dependencies

** redis-rb 0.03 By the excellent Ezra Zygmuntowicz It's best to install the version of the redis gem that is available in the client-libraries dir of the redis distribution. ** extlib, dependency for dm-core ** data-objects, dependency for dm-core ** dm-core next branch

Install

Run the following if you haven't already: gem sources -a http://gems.github.com Install the gem(s): sudo gem install whoahbot-dm-redis
